# 깃허브 폴더생성

1. 'Add file'의 'Create new file'를 클릭

2. 'Name your file...'에 생성할 폴더 이름을 적는다.

3. '/'를 입력한다. 입력하면 아래와 같이 새로운 폴더 구조가 생성

4. 임의의 파일을 1개 생성

5. commit
